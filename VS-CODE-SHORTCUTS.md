# VS Code Shortcuts Cheat Sheet

## The Essentials (Start Here)
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + Shift + P` | **Command Palette** — search for ANY action | The "search bar for everything" |
| `F11` | Toggle Full Screen on/off | Same as a browser |
| `Ctrl + `` ` `` | Toggle Terminal panel open/close | Quick access to command line |
| `Ctrl + B` | Toggle Sidebar (Explorer) on/off | Show/hide file tree |
| `Ctrl + S` | Save current file | Same as everywhere |
| `Ctrl + Z` | Undo | Same as everywhere |

## File & Navigation
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + P` | **Quick Open** — search for a file by name | Like Ctrl+P print... but for files |
| `Ctrl + Tab` | Switch between open tabs | Like Alt+Tab for browser tabs |
| `Ctrl + W` | Close current tab | Same as closing a browser tab |
| `Ctrl + Shift + E` | Focus the Explorer (file tree) sidebar | E = Explorer |
| `Ctrl + G` | Go to a specific line number | G = Go |

## Editing
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + D` | Select the next matching word | D = Duplicate selection |
| `Ctrl + /` | Comment/uncomment selected lines | The slash = comment slash |
| `Alt + Up/Down` | Move the current line up or down | Drag a line without cutting |
| `Ctrl + Shift + K` | Delete entire line | K = Kill line |
| `Ctrl + L` | Select entire line | L = Line |
| `Shift + Alt + Down` | Duplicate line below | Copy-paste a line instantly |

## Search
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + F` | Find in current file | F = Find (same as browser) |
| `Ctrl + H` | Find and Replace in current file | H = swap (H)ere |
| `Ctrl + Shift + F` | Search across ALL files in project | Global find |

## VS Code Panels
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + Shift + P` | Command Palette (do anything) | **Most important shortcut** |
| `Ctrl + ,` | Open Settings | Comma = configure |
| View → Extensions | Open Extensions panel | Install/manage extensions |
| View → Reset Layout | Reset everything back to default | Panic button for layout |

## Copilot / AI
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + I` | Open Copilot inline chat (in editor) | I = Inline |
| `Ctrl + Shift + I` | Open Copilot Chat panel | The AI assistant |
| `Tab` | Accept Copilot suggestion | Same as autocomplete |
| `Esc` | Dismiss Copilot suggestion | Nevermind |

## Terminal
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `Ctrl + `` ` `` | Toggle terminal open/close | Quick terminal access |
| `Ctrl + Shift + `` ` `` | Create new terminal | Fresh terminal tab |

## Debugging
| Shortcut | What It Does | Think of it as... |
|---|---|---|
| `F5` | Start debugging | Same as Visual Studio |
| `F9` | Toggle breakpoint on current line | Set a stop point |
| `F10` | Step over (next line) | Walk through code |
| `F11` | Step into a function | Go deeper into code |
| `Shift + F5` | Stop debugging | Kill debug session |

---

> **Tip:** You really only need 5 shortcuts to start:
> 1. `Ctrl + Shift + P` — Command Palette (find any action)
> 2. `Ctrl + P` — Quick open a file
> 3. `Ctrl + S` — Save
> 4. `Ctrl + `` ` `` — Toggle terminal
> 5. `Ctrl + B` — Toggle sidebar
>
> Everything else you'll pick up naturally as you use VS Code more.
